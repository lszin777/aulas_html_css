ID e CLASS

CLASS

Existem duas características principais do class

  • É possivel utilizar o mesmo class em varios elementos diferentes
  • É possivel utilizar várias classses no MESMO elemento

ID

Existem duas características pricipais do ID que são contrastantes com as características do class

  • Não é possivel utilizar o mesmo ID em vários elementos diferentes.
  • Não é possivel utilizar VÁRIOS IDs no mesmo elemento
